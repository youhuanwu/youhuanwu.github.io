## Purpose

The project delivered a fully automated document printing platform integrated with  ERP systems, supporting millions of inspection and quality certificates annually and ensuring regulatory compliance in large-scale pharmaceutical logistics.

## Key Features

Automatically check for new print tasks on schedule

## Challenges

The ERP system was developed with an older Builder 7 framework, and some controls are difficult to locate due to missing distinct attributes, requiring innovative solutions.

## Implementation

The main framework uses visual low-code automation software, combined with Python automation libraries.
